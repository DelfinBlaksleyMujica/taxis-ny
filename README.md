# 🚕 NY Taxis Analysis & Prediction

An end-to-end Data Science and Machine Learning project analyzing and predicting New York City taxi ride patterns. This repository contains the source code for our interactive web application, predictive API, and data processing pipelines, developed as a final project for the Le Wagon Data Science bootcamp.

## 🚀 Project Overview

### Key Features
* **Interactive Map Interface:** A user-friendly web application to visualize geographic data and model predictions in real-time.
* **Predictive API:** A dedicated backend service serving our trained Machine Learning models.

## 📂 Repository Structure

* `api/`: Contains the backend logic and routing for the REST API.
* `frontend/`: Code for the user interface and interactive map.

## 🛠️ Technologies Used

* **Language:** Python
* **Data Processing & ML:** Pandas, Scikit-Learn, Facebook prophet
* **Backend:** FastAPI, Uvicorn.
* **Frontend:** Streamlit & Folium for map visualization.
* **Environment & Tools:** Git, Make, Virtual Environments
